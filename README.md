# CareCo

CareCo is a modern, premium healthcare application designed to connect patients with care teams.

## Features
- **Patient Dashboard**: Real-time medication tracking, adherence stats, and assigned caller info.
- **My Caller**: View assigned care team member, call history, and one-tap calling.
- **Medications**: Track daily schedule (morning/afternoon/night) and view weekly adherence charts.
- **Health Profile**: View current conditions, allergies, medical history, and emergency contacts.
- **Settings**: Manage profile, view subscription plan, and update emergency contact.

## Tech Stack
- **Frontend**: React Native, Expo, React Navigation, Lucide Icons, Expo Linear Gradient
- **Backend**: Node.js, Express, MongoDB, Mongoose, Supabase Auth
